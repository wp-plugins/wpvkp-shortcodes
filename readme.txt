WPVKP Shortcodes is a collection of modern, responsive and attractive shortcodes which you can use in your wordpress website.

== Description ==
WPVKP shortcodes is an easy to use and customizable shortcode plugin. It has modern, mobile responsive and useful shortcodes which you can use in your posts and pages to make them more attractive and useful to your visitors or customers.

This plugin is developed by [WPVKP - WordPress Community Support](http://wpvkp.com). We always try to provide WordPress community with premium, professional and quality plugin and themes.

It will be regularly updated and will be maintained; obviously with all your help.

= How to use this shortcode plugin: =
Its preety easy. Lets say you want to add a button to your post or page. Then you will have to click on the WPVKP Shortcode option in your post editor and have to select button and it will automatically add the correct shortcode in your post. It will look something like this.

[wpvkp_button size="medium" color="violet modern" style="normal" link="http://wpvkp.com" target="_blank" rel="nofollow"]Preview[/wpvkp_button]

Now you can modify the button color by entering grey, voilet, blue, black, red, green or orange inside double quotes of color="".Example color="orange".
You can also switch to modern button style by just entering modern beside the color. Example color="voilet modern".

== Installation ==
1. Download the plugin from here.
2. Unzip the folder into the `/wp-content/plugins/` directory.
3. Activate the plugin through the \'Plugins\' menu in WordPress.
